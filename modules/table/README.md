# @workplus-css/table

### 安装

```bash
npm install @workplus-css/table
```

### 使用

仅提供*w-table*类名，只对table的合法元素进行修饰：

```html
 <div class="w-block">
  <table class="w-table">
    <thead>
      <tr>
        <th>ID</th>
        <th>Name</th>
        <th>Age</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>1</td>
        <td>Hejx</td>
        <td>18</td>
      </tr>
      <tr>
        <td>1</td>
        <td>Hejx</td>
        <td>18</td>
      </tr>
      <tr>
        <td>1</td>
        <td>Hejx</td>
        <td>18</td>
      </tr>
    </tbody>
  </table>
</div>
```
