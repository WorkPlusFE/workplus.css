# @workplus-css/button

### 安装

```bash
npm install @workplus-css/button
```

### 按钮类型

```html
<button class="w-btn" type="button">Default Button</button>
<button class="w-btn w-btn-primary" type="button">Primary Button</button>
<button class="w-btn w-btn-info" type="button">Info Button</button>
<button class="w-btn w-btn-success" type="button">Success Button</button>
<button class="w-btn w-btn-warning" type="button">Warning Button</button>
<button class="w-btn w-btn-error" type="button">Error Button</button>
```

### 块级按钮

```html
<a href="javascript:;" class="w-btn w-btn-block">Default Button</a>
<a href="javascript:;" class="w-btn w-btn-block w-btn-primary">Primary Button</a>
<a href="javascript:;" class="w-btn w-btn-block w-btn-info">Info Button</a>
<a href="javascript:;" class="w-btn w-btn-block w-btn-success">Success Button</a>
<a href="javascript:;" class="w-btn w-btn-block w-btn-warning">Warning Button</a>
<a href="javascript:;" class="w-btn w-btn-block w-btn-error">Error Button</a>
```

### 不可用按钮

```html
<button class="w-btn w-btn-block" disabled type="button">Disabled Button</button>
<button class="w-btn w-btn-primary" disabled type="button">Disabled Button</button>
<button class="w-btn w-btn-error" disabled type="button">Disabled Button</button>
```
